pd4ml
Tag footnote

PDF Footnote Tag. Forces generation of a footnote at the bottom part of the current page. The tag content goes to the footnote area prefixed with an incremented footnote counter value. The counter value (as a reference) is also appears at the original location of the footnote tag.Important: If the footnote content does not fit the remaining page space, it is wrapped to the next page. If the footnote content does not fit entire single page, the footnote does not print at all.
